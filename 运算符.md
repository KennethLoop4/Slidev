# 算术运算符

<div v-click=1>

算术运算符用于算术运算，返回值的类型为数值型


| 运算符 | 作用              | 示例          |
|-----|-----------------|-------------|
| +   | 求两个数的和          | 5+9=14      |
| -   | 求两个数的差：求一个数的相反数 | 8-5=3；-3=-3 |
| *   | 求两个数的积          | 6*5=30      |
| /   | 求两个数的商          | 5/2=2.5     |
| \   | 整除（两数相除取商的整数）   | 5\2=2       |
| ^   | 指数运算（求一个数的次方）   | 5^3=555=125 |
| Mod | 求模运算（两数相除取余数）   | 12 Mod 9=3  |


</div>
