# 变量






```vb {hide|hide|1|1,2|4|4,5|6|6,7|9,10|all} twoslash
'强制申明变量
Option Explicit
Sub test（）
    '变量的声明
    Dim a as String
    '变量的赋值
    a =”我是变量一！”
    MsgBox a
    '变量的赋值
    a =”我是变量二！”
    MsgBox a
End Sub
```

# 数据类型
